# Chrome Bookmarklet

A simple, customizable Chrome Bookmarklet project.

## Features
- **Easy Installation**: Drag-and-drop installation via a provided HTML page.
- **Non-blocking Notifications**: Uses a toast notification system instead of blocking alerts.
- **Customizable**: Easy to modify the JavaScript logic in `bookmarklet.js`.

## Installation
1. Open `index.html` in your browser.
2. Drag the "My Bookmarklet" button to your bookmarks bar.
3. Click the bookmark on any page to see it in action.

## Development
1. Modify `bookmarklet.js` with your desired logic.
2. Update `index.html` with the new URI-encoded JavaScript.
   - *Tip: You can use an online tool or a build script to minify and encode your JavaScript.*
